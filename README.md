The Power Monitoring and Consumption System is a web-based application designed to monitor, record, and analyze electricity consumption. The system helps users track energy usage, manage power consumption, and generate reports to support efficient energy management and cost reduct.
Objectives
Monitor electricity consumption in real time.
Store and manage power usage data.
Generate energy consumption reports
Improve decision-making for energy management.
Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Version Control: Git and GitHub
